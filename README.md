# Criando minha primeira API com FastAPI & Python

## Passo 1: Clone o Repositório

```bash
git clone https://github.com/ArieviloDavidson/fastApi.git
```
```bash
cd fastApi
```

## Passo 2: Instale as dependências
- Opcional - Criar ambiente virtual

```bash
pip install requirements.txt
```

## Passo 3: Execute o aplicativo

```bash
uvicorn main:app --reload
```

## Passo 4: Teste o aplicativo

Segure Ctrl e clique em
```bash
http://127.0.0.1:8000
```
No terminal

- Observe a documentação digitando ***/docs*** na url!

Bons estudos!
